核心 7 色调色板预览

Crust — 主背景色
#042326
用途：作为整个应用窗口最底层的背景（面板之间的“缝隙”）。
Base — 面板背景色
#0A3A40
用途：主要 UI 面板的背景色，如分区面板、详情面板、主列表等。
Surface — UI 元素色
#0F5959
用途：多数非交互或默认状态的 UI 元素，如按钮默认背景、表格斑马纹、滚动条滑块等。
Overlay — 悬停/交互色
#1D7373
用途：鼠标悬停在列表项、按钮、菜单项等可交互元素上时的高亮色。
Accent — 强调/选中色
#107361
用途：选中状态颜色，如列表中被选中的项目、按钮按下时的颜色。
Main Text — 主文本色
#E0FBFC
用途：绝大多数常规文本、标题、标签。
Sub Text — 次要文本色
#89D9D9
用途：辅助性、不太重要的文本，如表格表头文字、分区标题等。
